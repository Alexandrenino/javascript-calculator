# 🧮 JavaScript Calculator

Simple calculator built with HTML, CSS, Bootstrap and JavaScript.

This project was developed as part of a JavaScript course and implemented by me as a learning exercise.

## 🚀 Live Demo

https://alexandrenino.github.io/javascript-calculator/

## 📌 Features

- Basic arithmetic operations (+, -, *, /)
- Interactive buttons
- Keyboard support
- Backspace deletes the last character
- Clear button (C)

## 🛠 Technologies

- HTML
- CSS
- Bootstrap
- JavaScript

## 📚 What I Learned

- DOM manipulation
- JavaScript functions
- Event handling
- Keyboard events
- Basic calculator logic
